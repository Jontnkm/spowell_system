[평가점수 기준]

1. 지도한 선수 중 하나의 대회에서 평가점수가 10점 이상인 메달을 획득한 선수를 2명 이상 배출 -> 각 선수의 평가점수를 누적 합산하여 산출

2. 장애체육인 분야 지도자는 동일 종목에서 국제장애인올림픽위원회가 정한 장애유형에 따라 세부 종목이 증가한 경우 -> 해당 평가점수를 출전한 세부종목 수로 나누어 산출

3. 지도한 선수가 하나의 대회에서 다음 어느 하나에 해당하는 경우에는 가장 높은 평가점수 메달 1개 분의 금액을 전액 지급하고, 추가분에 대하여는 해당 금액의 2분의 1을 감액하여 지급
- 개인경기에서 2개 이상의 메달 획득
- 단체경기(동일 선수·인원으로 구성된 경기)에서 2개 이상의 메달 획득

4. 입상한 선수를 지도한 지도자의 수에 따라 다음과 같이 지급
- 지도자가 1명인 경우 -> 전액 지급 - 지도자가 2명 이상인 경우 -> [개인경기] 지도자 1명에 해당되는 금액 균등 배분, [단체경기] 지도자 2명에 해당되는 금액 균등 배분
